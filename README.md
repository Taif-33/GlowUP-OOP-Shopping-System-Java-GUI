# GlowUP-OOP-Shopping-System-Java-GUI-
GlowUP is a Java-based Object-Oriented Programming (OOP) project that simulates an online clothing company. It provides a graphical user interface (GUI) where users can create accounts, browse products, add items to a shopping bag, and complete purchases.

## Overview
GlowUP represents a full simulation of an online shopping system using OOP principles.
The project models a real-world company with multiple classes such as customers, employees, products, orders, and payment methods.

It demonstrates how object-oriented design can be applied to build a functional and interactive application.

## Features
### User Features
- Create account with input validation
- Browse available products
- Add items to shopping bag
- Update product quantity
- Remove products from cart
- Proceed to checkout

### Payment System
- Multiple payment methods:
- Credit Card
- Cash
- Payment validation and processing

### Shopping System
- Shopping bag management
- Order placement
- Shipping handling

### GUI Interface
- Java GUI-based application
- Interactive buttons and input dialogs
- Error handling (e.g., invalid email input)

## How It Works
### Create Account:
- User enters name, email, and password
- System validates input
### Browse Products:
- Products are displayed with prices
- User selects items
### Add to Bag:
- User chooses quantity
- Items are added to shopping bag
### Checkout:
- User selects payment method
- Order is processed
### Shipping:
- Order is prepared and shipped

## System Design (UML)
### Actors:
- Customer
- Employee
### Main actions:
- Create account
